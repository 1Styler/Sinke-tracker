# Vipps, så skylder du meg! 💸
### The Ultimate Sinke-Tracker™

Hold styr på hvem som skylder deg penger — med litt humor og frekke purremeldinger.

## Slik legger du dette på GitHub Pages

1. Gå til [github.com](https://github.com) og logg inn
2. Trykk **«+»** → **«New repository»**
3. Gi det navnet `sinke-tracker`, sett til **Public**
4. Trykk **«Create repository»**
5. Trykk **«uploading an existing file»**
6. Last opp **alle filene** fra denne zip-filen (index.html, manifest.json, og icons-mappen)
7. Trykk **«Commit changes»**
8. Gå til **Settings → Pages → Source: Deploy from branch → main → / (root)**
9. Trykk **Save**

Etter 1–2 minutter er appen live på:
`https://DITTBRUKERNAVN.github.io/sinke-tracker`

## Legg til på hjemskjermen (iPhone)

1. Åpne URL-en i **Safari**
2. Trykk **Del-ikonet** (firkant med pil opp)
3. Velg **«Legg til på hjemskjerm»**
4. Trykk **Legg til**

Appen åpner seg da i fullskjerm med eget ikon — akkurat som en ekte app! 📱
